Campus Resource Reservation API
Project Description
This is a backend API system for managing campus resource reservations. The system allows users to reserve campus resources such as study rooms, lab spaces, and equipment through a RESTful API.
Technologies Used

Node.js - JavaScript runtime environment
Express.js - Web application framework
MySQL - Relational database management system
GitHub - Version control and repository hosting

How to Run the Server Locally
Prerequisites

Node.js installed on your machine
npm (comes with Node.js)

Setup Instructions

Clone this repository
Navigate to the project directory
Install dependencies:

bash   npm install

Start the server:

bash   npm start

Test the server by visiting: http://localhost:3000/health

You should see a JSON response confirming the server is running.
